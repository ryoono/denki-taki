# 🔥 電気焚火キャンドル ― 設計思想（要約）

焚き火の魅力は「同じ瞬間が二度とない」こと。  
本作品はその偶然性と癒しを、電子制御ではなく**物理現象**によって再現する。  
ソレノイドが実際に空気を震わせ、吊り板が微妙に揺らぐ。  
その**わずかな誤差や乱れ**が、電子機器に“一期一会”の**わびさび**を与える。  
焚き火や焚き火キャンドルは火を用いるため扱いに注意が必要だが、本デバイスは電気で動作するため、安全性が高い。  


# 🔥 5W1H概要

| 要素 | 内容 |
|------|------|
| **Who（誰が）** | 室内で火を使えない人々（マンション・学生寮・家庭内など）や、就寝前に落ち着いた時間を過ごしたい人。 |
| **What（何を）** | 焚き火キャンドルのような**穏やかな炎の光と“パチパチ音”**を、火を使わずに再現する**リラックス用インテリアデバイス**。LEDの炎表現と、ソレノイドによる物理音が組み合わされた“電気焚火”。 |
| **When（いつ）** | 就寝前や読書・休憩・瞑想時など、**静かな夜のリラックスタイム**。長時間の点灯や寝落ちも想定し、安全性を重視。 |
| **Where（どこで）** | **室内（寝室・書斎・リビング）** など、火気厳禁かつ静音性を求める空間。小型設計で机上・ベッドサイドなどに設置可能。 |
| **Why（なぜ）** | 既製の電気焚火は「スピーカーからの擬似音が人工的」という課題がある。<br>本作品では、**ソレノイドが物理的に音を鳴らす**ことで、より**自然で偶然性のある“生きた音”**を再現する。<br>火の危険を排除し、現代生活に適した“安全な焚き火体験”を提供する。 |
| **How（どのように）** | - **LEDの揺らぎ**で炎を再現（オレンジ＋温白、帽子型拡散カバーで柔らかい光）<br>- **吊り板（木製・プラスチック製）をソレノイドで叩く**ことで“パチッ”“コツッ”と自然な破裂音を生成<br>- 板が揺れるたびに**当たり所が変わるため、同じ音は二度と鳴らない**<br>- 熱や煙を出さず、**安全で長時間使用可能**<br>- **ネットワーク接続機能**を持ち、Webページ経由で設定を行う（物理ボタンを排除） |


# 🔥 要求仕様書

## 1. 機能要求（Functional Requirements）

| ID | 要求項目 | 要求内容 |
|----|-----------|-----------|
| FR-1 | 炎の表現機能 | LEDの明滅により、焚き火キャンドルの炎を模した**穏やかな光のゆらぎ**を表現する。光源は複数色LED（オレンジ・温白など）を用いる。 |
| FR-2 | 音の生成機能 | ソレノイドの機械的打撃によって、焚き火の「パチッ」「コツッ」という**物理的な破裂音**を生成する。スピーカーは使用しない。 |
| FR-3 | ランダム性の再現 | 吊り板（木・プラスチック）が揺れ、**打撃の当たり所が毎回異なる**ように設計する。電子的なノイズ生成ではなく、機械的偶然性で音の多様性を生む。 |
| FR-4 | 光と音の調和 | LEDのゆらぎとソレノイドの発音が**全体として焚き火のテンポ感**を再現する。不自然な連打や長無音が発生しない。 |
| FR-5 | 安全動作 | 火災・感電・過熱などの危険がない構造とし、安全な低電圧電源で動作する。 |
| FR-6 | 操作性 | 本体に複数の物理スイッチを設けず、**最小限の電源スイッチまたは無操作起動**を基本とする。 |
| FR-7 | 自動停止機能 | 一定時間後に自動消灯・休止を行い、寝落ち時でも安全に使用できる。 |
| FR-8 | ネットワーク設定・制御機能 | 自宅のネットワークに接続し、**Webブラウザから設定や操作ができる**こと。<br>・自動停止時間や光の明るさをWebページ上で変更可能。<br>・Wi-Fi設定はプログラム直書きでよい。<br>・本体外観のスタイリッシュさを優先するため、ボタンや表示を極力省略する。 |

---

## 2. 非機能要求（Non-Functional Requirements）

| ID | 要求項目 | 要求内容 |
|----|-----------|-----------|
| NFR-1 | 静音性 | 音は就寝環境で心地よく感じられる程度とし、不快なノイズや過大な衝撃音を発生させない。 |
| NFR-2 | 音質 | 音は乾いた破裂音として自然に聴こえること。余韻やこもりを感じさせないこと。 |
| NFR-3 | 光の質感 | LED光は柔らかく、直視しても眩しくない。帽子型拡散カバーで穏やかに。 |
| NFR-4 | 審美性 | **ボタンや表示を極力排除**し、シンプルで静かな外観とする。配線は背面にまとめ、生活空間に溶け込むデザインとする。 |
| NFR-5 | 耐久性 | 通常の使用環境下で、打撃や吊り構造の破損・絡みが生じない。 |
| NFR-6 | メンテナンス性 | 構造を簡素にし、吊り板・糸・チップなどの交換を容易に行える。 |
| NFR-7 | 信頼性 | 長時間運転しても誤作動や過熱が発生しない。 |
| NFR-8 | 感性価値 | 同じ音が二度と鳴らない「偶然性」を体験価値として設計に組み込む。 |
| NFR-9 | ネットワーク操作性 | 設定画面はスマートフォンやPCのブラウザから簡単に操作できること。特別なアプリを不要とし、操作の反応が快適であること。 |

---

## 3. 制約条件（Constraints）

| ID | 制約項目 | 内容 |
|----|-----------|------|
| C-1 | 火気・煙 | 火炎・煙・香料などを一切発生させない。 |
| C-2 | 電源 | **安全な低電圧電源**で動作させる。給電方式（AC / DC / USB等）は問わない。 |
| C-3 | 筐体寸法 | デスクやベッドサイドに設置可能な小型サイズとする。 |
| C-4 | 環境条件 | 一般的な室内環境で安定して動作すること。 |
| C-5 | 騒音制限 | ソレノイドの駆動音や筐体の共振が不快にならないよう防振構造を採用する。 |
| C-6 | 安全規格 | ショート・逆接・過熱・絶縁不良が起きない構造とする。 |
| C-7 | 製作制約 | 入手容易な電子部品・素材を使用し、特殊加工を必要としない。 |
| C-8 | ネットワーク | ネットワーク設定値（SSID・パスワード）はプログラム内に固定で保持する。通信は家庭内ネットワーク内で完結する。 |
